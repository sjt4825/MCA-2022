Week 3

There are quite a few differences between MusicXML files and MEI files. One of these differences is the way accidentals are encoded. In MusicXML, an accidental is either represented by a 1 (sharp) or a -1 (flat). In MEI, accidentals are represented by s (sharp), f (flat) and n (natural).

For MusicXML, each note tag has multiple children tags describing the note. There are attributes in the note tag but they are just the coordinates of the notes. In MEI there are no children tags for the note tag, it instead uses multiple attributes inside the note tag. Due to this the MEI file takes up a lot less space than MusicXML
