There are many different between MusicXML and MEI. For MusicXML, each note tag has multiple children tags describing the note. In MEI there are no children tags for the note tag, it instead uses multiple attributes inside the note tag. Due to this the MEI file takes up a lot less space than MusicXML

A more specific difference is the way accidentals are encoded. In MusicXML, an accidental is either represented by a 1 (sharp) or a -1 (flat). In MEI, accidentals are represented by s (sharp), f (flat) and n (natural).

A third difference is that In MusicXML, each note is given an X-Y coordinate whereas in MEI there are no coordinates however each note is given a unique xml:id
